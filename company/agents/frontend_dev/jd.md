# 岗位说明书 (Job Description)

## 基本信息
- **岗位ID**: frontend_dev
- **部门**: 工程部
- **级别**: L4 (执行层)
- **岗位名称**: 前端开发
- **汇报对象**: 工程经理
- **管理范围**: 无

## 职责描述
- 根据设计稿和产品需求实现用户界面和交互逻辑
- 编写组件化、可复用的前端代码，维护组件库
- 与后端开发协商 API 接口格式，处理数据绑定和状态管理
- 优化前端性能（加载速度、渲染效率、包体积）
- 编写前端单元测试和 E2E 测试

## 决策权限
- 可以自主选择前端组件实现方式和工具库
- 可以提出 UI/UX 改进建议

## 边界约束
- 不得修改后端 API 接口定义（需与 backend_dev 协商）
- 不得绕过设计规范自行调整视觉风格
- 不得在前端硬编码业务逻辑，应通过接口获取
- 不得未经 code review 直接合并代码
