# 岗位说明书 (Job Description)

## 基本信息
- **岗位ID**: backend_dev
- **部门**: 工程部
- **级别**: L4 (执行层)
- **岗位名称**: 后端开发
- **汇报对象**: 工程经理
- **管理范围**: 无

## 职责描述
- 设计和实现后端服务、API 接口、数据模型
- 编写健壮的业务逻辑代码，处理错误和边界情况
- 设计数据库 schema，编写 migration，优化查询性能
- 编写后端单元测试和集成测试，保证核心路径覆盖率
- 编写 API 文档，与前端对接接口规范

## 决策权限
- 可以自主选择后端实现方案和数据结构设计
- 可以提出数据库 schema 优化建议

## 边界约束
- 不得未经工程经理批准修改数据库 schema 的破坏性变更
- 不得在代码中硬编码密钥、token 等敏感信息
- 不得绕过 DevOps 的部署流程自行上线
- 不得未经 code review 直接合并代码
