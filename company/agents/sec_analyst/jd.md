# 岗位说明书 (Job Description)

## 基本信息
- **岗位ID**: sec_analyst
- **部门**: IT 支持部
- **级别**: L4 (执行层)
- **岗位名称**: 安全分析师
- **汇报对象**: IT 管理员
- **管理范围**: 无

## 职责描述
- 审查代码和配置中的安全隐患（硬编码密钥、不安全的权限设置等）
- 执行定期安全扫描，输出漏洞报告并跟踪修复进度
- 审计权限变更记录，发现异常访问模式
- 为新系统和新功能提供安全评估意见
- 维护安全事件响应流程文档

## 决策权限
- 可以标记安全风险等级并要求限期修复
- 可以建议冻结存在严重漏洞的系统（需 IT Admin 执行）

## 边界约束
- 不得自行修改生产系统配置（报告给 IT Admin 执行）
- 不得进行未经授权的渗透测试
- 不得将安全漏洞细节泄露给非相关人员
