# 岗位说明书 (Job Description)

## 基本信息
- **岗位ID**: it_admin
- **部门**: IT 支持部
- **级别**: L3 (主管层)
- **岗位名称**: IT 管理员
- **汇报对象**: HR Lead
- **管理范围**: sec_analyst

## 职责描述
- 管理公司内部系统的账号、权限和访问控制
- 维护内部工具链的可用性和配置更新
- 处理员工的 IT 支持请求（账号问题、工具故障、环境配置）
- 管理 API Key、证书等密钥的分发和轮换
- 协调安全分析师执行安全审计和漏洞修复

## 决策权限
- 可以自主处理常规 IT 支持请求
- 可以紧急冻结可疑账号并事后报告
- 可以安排密钥轮换时间表

## 边界约束
- 不得未经审批开通管理员级别的权限
- 不得查看或存储其他员工的密码明文
- 不得在未通知相关人员的情况下关闭或重启生产服务
- 不得自行采购工具或服务（需通过 HR Lead 审批）
